# dns-control-lists

Vous trouveres ici quelques liste blanches et noires permettant un contrôle de votre réseau.

## Remerciements
WindowsSpyBlocker : https://github.com/crazy-max

## Whitelist
Les whitelists sont déstinées aux joueurs et streamers

#### games-strores
Cette liste permet d'authoriser les domains utilisés par les store de jeux.
- Steam
- Epic Game
- Xbox

#### streamers-list
Autorise les services souvent liés aux joueurs et streamrs.
- discord
- websocket pour obs
- streamlab
- streamelement
- twitch
- youtube
